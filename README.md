# XYWSoundChanger
对录制或下载的视频进行行变声，可直接运行效果。
（使用了第三方库SoundTouch）

## 功能
1. 可以对视频文件、音频文件变声并另存新的文件。
2. 可以对sampleBuffer进行变声并返回处理后的sampleBuffer。

### 音频变声流程

1. 解码音频
2. 变声
3. 编码音频

### 视频变声流程

1. 从视频分离音频出音频文件
2. 解码音频
3. 音频变声
4. 编码音频
5. 用变声后的音频替换原视频中的音频

### smapleBuffer变声流程

1. 从buffer中获取pcm数据
2. 对数据进行变声处理
3. 将数据合成为sampleBuffer
